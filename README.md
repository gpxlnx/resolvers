## resolvers

List of Fresh DNS resolvers updates every 1 hour

- Resolvers updated within the last **1 hour** using `dnsvalidator` are saved in `resolvers.txt`.  
- Resolvers from repositories updated within the last **24 hours** are saved in `stable_resolvers.txt`.  
- All resolvers from repositories, regardless of whether they were updated in the last 24 hours, are saved in `unstable_resolvers.txt`.
